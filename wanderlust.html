Wanderlust

Branding, Print Design, Layout

Wanderlust was an assignment for Digital Grid Systems class in school. The assignment was to create an identity for a magazine and create a magazine layout for an article surrounding a theme that the magazine was based around. I love travelling and nature. For some time now, I've been in desperate need for a vacation and I'd been feeling what some call wanderlust. It was this desire that inspired the identity.
